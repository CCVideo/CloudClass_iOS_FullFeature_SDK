2 ## 1.概述
3 CC基础版SDK 是一个适用于iOS平台的云课堂SDK，使用此SDK可以与CC视频云服务进行对接，在iOS端使用CC视频的云课堂功能。
4 
5 ### 1.1 功能特性
6 | 功能 | 描述 | 备注 |
7 | :-------- | :----------------------- | :--- |
8 | 推流 | 支持推流到服务器 | |
9 | 拉流 | 支持从服务器订阅流 | |
10 | 获取流状态 | 支持获取流的状态(发报数、收报数、丢包数、延时) | |
11 | 前后摄像头切换 | 支持手机前后摄像头切换 | |
12 | 后台播放 | 支持直播退到后台只播放音频 | |
13 | 支持https协议 | 支持接口https请求 | |
14 
15 ### 1.2 阅读对象
16 本文档为技术文档，需要阅读者：
17 
18 * 具备基本的iOS开发能力
19 
20 * 准备接入CC视频的云课堂SDK相关功能
21 
22 ### 1.3 SDK架构
23 * 支持的CPU架构有armv7,arm64
24 
25 * 支持的最低系统版本iOS8

110 
111 ```objc
112 //视频流add回调
113 #define WS(weakSelf) __weak __typeof(&*self)weakSelf = self;
114 #pragma mark - 流
115 - (void)onServerDisconnected
116 {
117  [self.streamerBasic leave:^(BOOL result, NSError *error, id info) {
118 
119  }];
120  WS(ws);
121  dispatch_async(dispatch_get_main_queue(), ^{
122  //退出当前控制器
123  [ws.navigationController popViewControllerAnimated:NO];
124  });
125 }
126 
127 - (void)onStreamAdded:(CCStream*)stream
128 {
129  if ([stream.userID isEqualToString:self.stremer.userID])
130  {
131  //自己的流不订阅
132  self.localStream = stream;
133  return;
134  }
135  if (stream.type == CCStreamType_Mixed)
136  {
137  self.mixedStream = stream;
138  }
139  //订阅视频流
140  [self autoSub:stream];
141 }
142 
143 - (void)onStreamRemoved:(CCStream*)stream
144 {
145  if ([stream.userID isEqualToString:self.stremer.userID])
146  {
147  //自己的流没有订阅
148  return;
149  }
150  [self autoUnSub:stream];
151 }
152 
153 - (void)onStreamError:(NSError *)error forStream:(CCStream *)stream
154 {
155  CCLog(@"%s__%d__%@__%@", __func__, __LINE__, error, stream.streamID);
156 }
157 
158 - (void)autoSub:(CCStream *)stream
159 {
160  WS(ws);
161  [self.stremer subcribeWithStream:stream qualityLevel:0 completion:^(BOOL result, NSError *error, id info) {
162  if (result)
163  {
164  [weakSelf.streamView showStreamView:info];
165  }
166  else
167  {
168  [weakSelf showError:error];
169  }
170  }];
171 }
172 ```

311 ## 4.功能使用
312 ### 4.1 预览
313 预览是将初始化相机的流渲染出来：
314 ```objc
315 - (void)startPreview:(CCComletionBlock)completion;
316 ```
317 
318 ### 4.2 开始直播
319 点击开始直播，成功以后，进行推流和拉流操作：
320 * 开始直播方法
321 ```objc
322 - (BOOL)startLive:(CCComletionBlock)completion;
323 ```
324 ### 4.3 结束直播
325 结束当前直播：
326 * 结束直播方法
327 ```objc
328 - (BOOL)stopLive:(CCComletionBlock)completion;
329 ```
330 ### 4.4 推流/取消推流
331 
332 推本地相机的流到服务器：
333 * 推流的方法
334 ```objc
335 - (BOOL)publish:(CCComletionBlock)completion;
336 ```
337 
338 取消推相机本地流到服务器：
339 * 取消推流的方法
340 ```objc
341 - (BOOL)unPublish:(CCComletionBlock)completion;
342 ```
343 ### 4.5 拉流/取消拉流
344 
345 从服务端拉流：
346 * 拉流方法
347 ```objc
348 - (BOOL)subcribeWithStream:(CCStream *)stream qualityLevel:(int)level completion:(CCComletionBlock)completion;
349 ```
350 取消从服务端拉流：
351 * 取消拉流方法
352 ```objc
353 - (BOOL)unsubscribeWithStream:(CCStream *)stream completion:(CCComletionBlock)completion;
354 ```
355 ### 4.6 添加RTMP推流/取消RTMP推流
356 
357 添加RTMP流到服务端：
358 * 添加RTMP流方法
359 ```objc
360 - (BOOL)addExternalOutput:(NSString*)url completion:(CCComletionBlock)completion;
361 ```
362 
363 取消RTMP流到服务端：
364 
365 * 取消添加RTMP流方法
366 ```objc
367 - (BOOL)removeExternalOutput:(NSString *)url completion:(CCComletionBlock)completion;
368 ```
369 
370 ### 4.7 切换摄像头
371 
372 切换摄像头，前置摄像头和后置摄像头：
